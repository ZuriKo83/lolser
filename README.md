# 내전서버 이용 가이드

내전서버 게임을 위한 커뮤니티 공간입니다.  
서버 규정을 지키며 자유롭게 이용해 주세요.

---

## STEP 1. 모든 채널 표시

서버를 이용하려면 모든 채널 표시 기능을 활성화해야 합니다.  
해당 기능이 비활성화된 경우, 서버 내 중요 카테고리를 확인할 수 없습니다.

## STEP 2. 닉네임 변경

컨텐츠 이용을 위해 디스코드 닉네임을 양식대로 변경해야 합니다.  
포지션 및 티어 기재가 필수이며, 아래 형식을 참고하세요.

### 양식
닉네임 / 롤 닉네임#태그 / 티어

### 예시
나미 / 나 미#최고 / D4

###탑레이팅###이 잘못 되어있는 경우, 최대 차단까지 이어질 수 있습니다.
본인명의 계정이 아닌 타명의 계정이여도 본인이 플레이 하였으면 그 계정으로 기재해야합니다.
닉네임이 기준과 맞지 않거나 확인이 어려운 경우, 관리자가 수정 요청을 할 수 있습니다.

## STEP 4. 서버 이용 용도

### 1. 게임 구인 채널
- 롤파티모집 - 리그 오브 레전드 파티원을 모집하는 채널입니다.
- 타게임모집 - 다른 게임의 차티원을 모집하는 채널입니다.

  게임명과 모집 내용을 상세히 기재해 주시기 바랍니다.

### 2. 경고 채널
 본 서버는 경고 시스템이 4회 누적시 서버 영구 블랙 시스템입니다.
 
 ###[내전규정](/내전규정.md) 참고
- 내전경고 - 내전 도중 규정을 위반한 유저에게 내전 경고가 부여됩니다.
- 서버경고 - 서버 규정을 위반한 이용자에게 서버 경고가 부여됩니다.
- 내전블랙 - 지정된 기간 동안 내전에 참여할 수 없습니다.
- 서버블랙 - 서버 접근이 영구적으로 차단됩니다.


### 3. 기타 안내
- 부적절한 언행 및 게시물은 삭제될 수 있습니다.
- 반복 위반 시 제재가 적용됩니다.
- 모든 이용자는 서버 규정을 준수해야 합니다.

---

공용 커뮤니티 특성상 관리자의 판단에 따라 게시물 삭제 및 제재가 이루어질 수 있습니다.
